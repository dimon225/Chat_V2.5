This view is the server Connections tab:

 

This tab offers the administrator multiple choices:

User's List: its the list of all connected clients

Admin Chat: It's a reserved area for adminitrator chat with the clients in the public chat

The Admin chat is accompanied by three buttons and one text field

Clear Chat button: will clear the admin chat text area

Show Chat: will show chat to the administrator and change its text to hide chat by clicking hide chat the chat will be hidden from the adminstrator

Broadcast button: will broadcast to all users the message typed in the text field that's on the right

The text field is used to public chat with the users.

The User's List is accompanied by three buttons and one text field

These components are disabled until the adminstrator has chosen a user in the list

Send Message: will send a private message to that user.

Disconnect: will disconnect a user and close its application

Ban user: will put the user name and black list

and never let him return as long as the server is online if server restarts however the black list is not saved so the banned user may return again

 

This view is the server Communications tab:

This tab offers some cool features that you can do, to clients

The list on the right will hold all connected clients

as soon as you click on one name the possiblities are endless.

The List on the right will hold files paths of the chosen client basically when the adminstrator clicks

Get All Client's Files: the request will be sent to the remote machine that has a potential to scan the directories fill the list and send it back to the server and the files will be displayed in the list.

Steal file: if the adminstrator sees the file he likes he simply clicks on it and presses this button this will get the file to the server client will not notice.
